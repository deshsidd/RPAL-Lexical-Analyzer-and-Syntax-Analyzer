INTRODUCTION
------------
Made a scanner and parser for RPAL Programming language using c++.



USAGE
-----
- To Compile:

>> make


- Usage: ./p1 [-ast] [-l] filename

-ast generates  Abstract Syntax Tree
-l generates the input
